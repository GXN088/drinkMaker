# drinkMaker
В этой задаче тебе нужно:

В отдельном файле создать класс DrinkMaker с тремя абстрактными методами:
void getRightCup() - выбрать подходящую чашку;
void putIngredient() - положить ингредиенты;
void pour() - залить жидкость.
В классе DrinkMaker создать и реализовать метод void makeDrink(), который готовит напиток в такой последовательности: выбирает чашку, кладет ингредиенты, заливает жидкость.
В отдельных файлах создать классы LatteMaker и TeaMaker, которые наследуются от DrinkMaker.
Распределить следующие фразы между всеми методами в классах LatteMaker и TeaMaker:
"Заливаем кипятком"
"Берем чашку для латте"
"Насыпаем чай"
"Берем чашку для чая"
"Заливаем кофе"
"Делаем молоко с пенкой"
Каждый метод должен выводить в консоль свою фразу, не пересекаясь с другими методами.

Псевдокод для данной задачи (без привязки к конкретному языку, описывающий логику работы):

---

Абстрактный класс DrinkMaker

```
абстрактный класс DrinkMaker:
    // Шаблонный метод (алгоритм приготовления напитка)
    метод makeDrink():
        getRightCup()   // выбрать чашку
        putIngredient() // положить ингредиенты
        pour()          // залить жидкость

    абстрактный метод getRightCup()
    абстрактный метод putIngredient()
    абстрактный метод pour()
```

---

Класс TeaMaker (наследует DrinkMaker)

```
класс TeaMaker расширяет DrinkMaker:
    
    метод getRightCup():
        вывести "Берем чашку для чая"
    
    метод putIngredient():
        вывести "Насыпаем чай"
    
    метод pour():
        вывести "Заливаем кипятком"
```

---

Класс LatteMaker (наследует DrinkMaker)

```
класс LatteMaker расширяет DrinkMaker:
    
    метод getRightCup():
        вывести "Берем чашку для латте"
    
    метод putIngredient():
        вывести "Делаем молоко с пенкой"
    
    метод pour():
        вывести "Заливаем кофе"
```

---

Точка входа (Solution)

```
метод main():
    создать объект latteMaker = новый LatteMaker()
    создать объект teaMaker = новый TeaMaker()

    вызвать latteMaker.makeDrink()
    вызвать teaMaker.makeDrink()
```

Ожидаемый вывод при выполнении:

```

Делаем молоко с пенкой
Заливаем кофе
Берем чашку для чая
Насыпаем чай

```

Суть паттерна:
makeDrink() задаёт фиксированный алгоритм («скелет») приготовления, а конкретные шаги реализуются в наследниках. Это шаблонный метод (Template Method).
